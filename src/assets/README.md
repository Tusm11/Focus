# Assets

This directory contains all image assets for the Focus extension.

## Required Images

### Icons
- `icon-16.png` - 16x16 extension icon
- `icon-48.png` - 48x48 extension icon
- `icon-128.png` - 128x128 extension icon

### UI Assets
- `avatar-default.png` - Default player avatar
- `character-default.png` - Default character image
- `pet-default.png` - Default pet image

## Creating Assets

You can create these images using:
- Figma
- Adobe XD
- GIMP
- Online tools like Pixlr or Canva

### Recommended Dimensions
- Icons: 16x16, 48x48, 128x128 (PNG format)
- Avatar: 48x48 (PNG format)
- Character: 200x200 (PNG format)
- Pet: 120x120 (PNG format)

## Placeholder Generation

For development, you can generate placeholder images using online tools or create simple SVG placeholders.
